# ☀️ SunShare

> Sistema de Gestão e Compartilhamento de Créditos de Energia Solar em Microrredes Comunitárias.

Projeto acadêmico desenvolvido para a disciplina de Gestão e Qualidade de Software do IBMR.

---

## 📖 Sobre o Projeto

O SunShare é uma aplicação desenvolvida em Java que simula o gerenciamento de créditos de energia solar entre usuários de uma comunidade.

O sistema permite registrar geração e consumo de energia, calcular saldos de créditos e realizar transferências entre participantes da microrrede.

A proposta está alinhada aos Objetivos de Desenvolvimento Sustentável (ODS):

- ⚡ ODS 7 – Energia Limpa e Acessível
- 🏙️ ODS 11 – Cidades e Comunidades Sustentáveis

---

## ✨ Funcionalidades

- 👤 Cadastro de usuários
- ☀️ Registro de geração de energia
- ⚡ Registro de consumo de energia
- 🔄 Transferência de créditos energéticos
- 📊 Relatório completo de usuários e transações
- ✅ Validação de nomes duplicados
- 🖥️ Interface gráfica desenvolvida com Java Swing

## 🚀 Como Executar

### Pré-requisitos

- Java JDK 17 ou superior
- Visual Studio Code, IntelliJ IDEA ou Eclipse

### Executando

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/SunShare.git
```

2. Abra o projeto na IDE.

3. Execute o arquivo:

```text
Main.java
```

4. A aplicação será iniciada com interface gráfica.
